ポートフォリオをご覧頂きありがとうございます。丹羽と申します。4ヶ月間職業訓練校に通い（うち1ヶ月は企業実習）、HTMLやCSS、jQuery、デザインなど、Webサイトを作る基礎を1から学びました。訓練終了後から今までの間に、企業様のLPを2点作らせて頂きました。その後、契約社員として大手美容予約アプリのUI/UXデザインや、画像のレタッチに関わってまいりました。まだまだ勉強中ではありますが、情報の優先度を考えた、基本的なデザインを作ることができます。

これまでの主な経歴として、某人材Webサービスの記事・画像の制作や、運用、進行管理、顧客折衝の経験があります。いずれの業務においてもヒアリングを重視し、お客様の想いや事業、課題等についてきちんと理解した上で採用活動や、広告活動のサポートを行うよう心がけてきました。今後もこの姿勢を大切に、また、更なるスキルアップを目指しながら、お客様とユーザーに寄り添う制作を行っていきたいと考えています。

SKILL

HTML5

手打ちで、Webサイトを一から構築するためのHTML記述が可能です。より分かりやすく、SEOを考慮したコードを書くことができるように勉強中です。

CSS3

レスポンシブを含む、基本的なCSSコーディングが可能です。色やサイズ、レイアウトなどを指定することができます。

VS Code

HTML、CSS、jQueryコーディングを行う際にはVisual Studio Codeを使用しています。

jQuery

静的Webサイトを動的にするための簡単なコードの記述や、プラグインの導入ができます。

Photoshop

写真の修正や色調補正、合成、切り抜きなどができます。また、レイヤーの仕組みを理解し扱うことができます。デザインカンプはXDやIllustratorで作ることが多いですが、Photoshopでの制作も可能です。

Illustrator

主にデザイン制作やロゴ、アイコンの作成に使用しています。ベジェ曲線などを使用して簡単なイラストを描くことが可能です。基本機能は問題なく使うことができます。

XD

デザインカンプ制作時には主にXDを使用しています。IllustratorやPhotoshopと同様、基本機能は問題なく使用することができます。

Figma

勉強中ですが、デザインやプロトタイプの共有など、基本的な操作を行うことができます。

PHP

訓練終了後に問い合わせフォームやログイン・ログアウト、ショッピングカートなどの作成を通して、仕組みを学びました。自分で実際に実装できるよう、引き続き勉強しています。

WordPress

訓練終了後にWordPressについて学び、固定ページと投稿ページの違いなど、基本的な仕組みを理解しました。記事を書いて投稿することができます。テーマのカスタマイズや作成などができるよう、スキルアップを目指して勉強中です。

Word

画像・図形を挿入した資料や、礼状・送付状など、社内外文書の作成ができます。レイアウトの調整も可能です。

Excel

表の作成ができます。前職ではExcelでスケジュール表を作り、進捗管理を行っていました。

PowerPoint

プレゼンテーション用のスライドショーの作成ができます。グラフや画像の挿入なども可能です。

Slack

就業後に使用する可能性を想定し、訓練生同士でコミュニケーションを取る際に活用していました。

Chatwork

LPを制作させて頂いた際、クライアントとの連絡ツールとしてChatworkを使用していました。

ABOUT

丹羽niwa

これまでに人材情報会社や広告代理店にて、顧客や営業と関わりながら、広告制作における進行管理やライティングも行っておりました。様々なお客様を担当させて頂く中で、お客様の顔をとなり、印象を大きく左右する、デザインや文章制作の重要性を強く感じました。そこで、お客様の魅力を伝える制作物を手がけたいと考えています。

お客様と接する際には、丁寧に寄り添う姿勢と、お客様について理解することを大切にしてまいりました。担当させて頂いたお客様からは「安心できた」「自分の事業の新たな魅力に気が付いた」とのお声を頂くことが多かったです。今後もこの姿勢を大切にしながら、お客様とユーザーを繋ぐWebサイトの制作に貢献していきたいと考えております。
